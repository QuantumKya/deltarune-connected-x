# Deltarune Connected X
Name WIP

### A desktop port of Deltarune Connected, playable on PC without Roblox.
Made on Roblox, Deltarune Connected is a multiplayer game with Deltarune scenes and characters, which allows for fun, silly interactions while pretending to be characters from Deltarune. This is a PC/Mac port of that Roblox game, skirting the need to use Roblox.  
I was inspired to make this because of Roblox's absolutely blunderful moderation policies and my personal feelings about sharing biodata with billion-dollar companies.
